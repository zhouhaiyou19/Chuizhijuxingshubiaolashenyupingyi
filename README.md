# C# 绘制矩形：鼠标拉伸与平移

## 简介

本资源文件提供了一个C#绘图源码，允许用户通过鼠标操作来绘制矩形，并且支持按住鼠标拖动图形以及通过八个方向拉伸图形。该源码适用于需要实现图形交互操作的C#开发项目。

## 功能特点

- **鼠标拖动**：用户可以通过按住鼠标左键拖动矩形，实现图形的平移操作。
- **八个方向拉伸**：用户可以通过鼠标在矩形的八个方向上进行拉伸操作，调整矩形的大小。
